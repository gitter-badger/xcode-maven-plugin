Full name:

ca.mestevens.ios:xcode-maven-plugin:0.6.0:xcode-package-framework

Description:

Goal to package the artifact(s).

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: package.
