Plugin Documentation

Goals available for this plugin:

Goal Description
xcode:framework-dependencies Goal which generates your framework dependencies in the target directory.
xcode:set-keychain Goal to set your keychain if needed.
xcode:xcode-build Goal to build your artifact.
xcode:xcode-package-app Goal to package your .app file and optionally deploy it to the simulator.
xcode:xcode-package-framework Goal to package the artifact(s).
xcode:xcode-test Goal to test your artifact(s).

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK Default target for maven-compiler-plugin version 2.5.1
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>ca.mestevens.ios</groupId>
          <artifactId>xcode-maven-plugin</artifactId>
          <version>0.7.2</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>ca.mestevens.ios</groupId>
        <artifactId>xcode-maven-plugin</artifactId>
        <version>0.7.2</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
